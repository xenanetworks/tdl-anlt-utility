==============================================
Xena ANLT Utility Documentation
==============================================

Xena AN/LT Utility provides an **interactive shell** for users to control Xena testers via **command-line interface** (CLI) commands to performance operations such as :term:`AN/LT` testing and resource management (more will be added in future releases) in an interactive fashion.
