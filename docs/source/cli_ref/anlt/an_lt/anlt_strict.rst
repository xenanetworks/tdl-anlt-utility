anlt strict
===========

Description
-----------

Enable/disable ANLt strict mode. If enable, errored frames will be ignored.


Synopsis
--------

.. code-block:: text
    
    anlt strict
    [--on/--off]


Arguments
---------


Options
-------

``--on/--off``

Should enable AN/LT strict mode, default to ``--on``.


Examples
--------

.. code-block:: text

    anlt-utility[123456][port0/2] > anlt strict --on
    Port 0/2 AN/LT strict mode: on

    anlt-utility[123456][port0/2] >
