anlt start
===================

Description
-----------

Apply and start AN/LT to the working port.

Apply the shadow configuration from :doc:`../an/an_config`, :doc:`../lt/lt_config`, and :doc:`../lt/lt_im` to the working port, and then start AN/LT on the port.


Synopsis
--------

.. code-block:: text
    
    anlt start
